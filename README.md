【游戏名称】
植物大战僵尸 - 坚果保龄球

【游戏介绍】
这是一款H5原生开发的休闲动作类小游戏，灵感来自经典游戏《植物大战僵尸》中的保龄球关卡。游戏中，你控制一颗坚果在草坪上左右滚动，撞击从上方不断涌来的僵尸。小心不要撞到两侧的沟渠，否则游戏就会结束！

【玩法说明】
游戏规则：
僵尸会从屏幕上方不断出现并向下移动，你需要控制坚果左右移动去撞击它们。随着游戏进行，僵尸出现速度会越来越快，难度逐渐提升。

道具说明：
1. 普通僵尸（绿色）- 最基础的僵尸，速度中等，一击即可消灭。
2. 路障僵尸（黄色）- 头戴路障锥，移动速度较快，需要更强的撞击力。
3. 铁桶僵尸（灰色）- 头戴铁桶，移动速度较慢但更加坚固。

操作方式：
【桌面端操作】
按左方向键或A键 - 向左移动坚果
按右方向键或D键 - 向右移动坚果

【移动端操作】
点击屏幕左侧按钮 - 向左移动坚果
点击屏幕右侧按钮 - 向右移动坚果

游戏目标：
撞击尽可能多的僵尸，获得更高的分数。千万不要撞到草坪两侧的沟渠（黑色区域），否则游戏立即结束！

Tips：
连续撞击僵尸可以获得连击加成，连击越高得分越多。坚果撞击僵尸后会反弹，利用好反弹效果可以连续击中多个僵尸。保持冷静，挑战最高分和最大连击数！

【技术介绍】
HTML5原生开发，主要语言是JavaScript，使用了Canvas 2D渲染技术，实现了流畅的物理碰撞效果和粒子系统，纯原生实现无需任何框架。
