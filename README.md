植物大战僵尸 - 坚果保龄球 - 游戏操作说明
================================================

【游戏简介】
复刻自《植物大战僵尸》中的经典迷你游戏“坚果保龄球”。在本游戏中，你将扮演一颗勇敢的坚果，在草坪上滚动并击退来袭的僵尸大军。利用物理碰撞的特性，在僵尸群中弹射得分！

【如何开始】
1. 找到游戏文件 "index.html"。
2. 双击文件，使用任意现代浏览器（如Chrome, Edge, Safari, Firefox）打开即可。
3. 点击屏幕上的“开始游戏”按钮。

【操作指南】

[电脑端 (PC)]
- 向左移动：按下键盘 "←" (左箭头) 或 "A" 键。
- 向右移动：按下键盘 "→" (右箭头) 或 "D" 键。
- 重新开始：游戏结束后，点击屏幕上的“重新冒险”按钮。

[移动端 (手机/平板)]
- 向左移动：按住屏幕左下方的 "⬅️" 按钮。
- 向右移动：按住屏幕右下方的 "➡️" 按钮。

【游戏规则】
1. 核心玩法：控制坚果左右滚动，撞击迎面而来的僵尸。
2. 物理弹射：撞击僵尸后，坚果会根据撞击角度发生反弹。
3. 死亡条件：
   - ⚠️ 警告：草坪两侧是水沟（墙壁），坚果如果撞到两侧边缘，游戏立即结束！
   - 需要利用撞击僵尸的反作用力来调整位置，避免滑入水沟。
4. 连击系统：
   - 短时间内连续撞击僵尸可以触发连击（Combo）。
   - 连击数越高，获得的额外分数越多。

【僵尸图鉴】
- 普通僵尸（绿色）：移动速度正常，一击必杀。
- 路障僵尸（黄色）：移动速度稍快，也是一击必杀（在H5版中简化）。
- 铁桶僵尸（灰色）：移动速度较慢，体型较大。

【高分技巧】
1. 控制惯性：坚果滚动有惯性，不要一直按住方向键，适时松手利用滑行。
2. 角度射击：尝试用坚果的侧面去蹭僵尸，产生横向的弹射力，更容易在僵尸群中来回弹跳。
3. 远离边缘：尽量保持在场地中央活动，避免因反弹失控撞墙。
